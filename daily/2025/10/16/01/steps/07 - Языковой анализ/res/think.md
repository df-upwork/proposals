Замечание: Предложение «The built-in Windows security policy... (CBA)» является длинным и сложным для восприятия из-за придаточного предложения в конце («which is used in Certificate-Based Authentication (CBA)»). Разделение этого предложения на 2 более коротких улучшит читаемость и ясность изложения.
Степень уверенности: 85/100.
Предложение:
The built-in Windows security policy «Interactive logon: Smart card removal behavior» works only with tokens using the smart card protocol (PIV).
This protocol is used in Certificate-Based Authentication (CBA).
