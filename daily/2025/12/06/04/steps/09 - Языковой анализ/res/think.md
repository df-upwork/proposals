# 1\. Замечания по пункту 5.1.2

Степень уверенности: 90
Статус неправоспособности в Калифорнии может быть не только «Suspended» (приостановлен), но и «Forfeited» (аннулирован). Процедура «Revivor» применима к обоим статусам, и для полноты картины следует упомянуть оба.
Предложение по устранению:

```markdown
5.1.2) `R2`: verify `C`’s status in `CA` and complete the «Revivor» process if the status is «Suspended» or «Forfeited».
```

# 2\. Замечания по пункту 5.1.3

Степень уверенности: 100
Ссылка на `Corp. Code § 1151` юридически некорректна для данной ситуации. Статья 1151 регулирует конвертацию иностранной сущности *в* калифорнийскую корпорацию (inbound). Для конвертации калифорнийской корпорации *в* иностранную сущность (outbound) применяется статья 1157.
Предложение по устранению:

```markdown
5.1.3) `R3`: execute «Domestication» into `PA` (15 Pa.C.S. § 371) followed by «Conversion» in `CA` (Corp. Code § 1157 or § 17710.02).
```

# 3\. Замечания по пункту 5.2

Степень уверенности: 90
Использование термина «resident» (резидент) в отношении корпорации не является стандартным для корпоративного права. После доместикации компания становится «domestic entity» (внутренним лицом) штата Пенсильвания. Термин «resident» чаще используется в налоговом контексте для физических лиц.
Предложение по устранению:

```markdown
`C` becomes a `PA` domestic entity cleared of past penalties via `VDP`.
```

```markdown
5.3.1) For corporations, `R` qualifies as a tax-free Type «F» reorganization (IRC § 368(a)(1)(F)).
```

# 5\. Замечания по пункту 5.3.2

Степень уверенности: 80
Слово «ensuring» (гарантируя) является слишком категоричным. Банки могут потребовать закрытия счетов из-за внутренних процедур compliance (KYC) даже при сохранении EIN. Более точный термин — «facilitating» (способствуя/облегчая).
Предложение по устранению:

```markdown
5.3.2) `C` retains the federal EIN and history, facilitating continuity for banking and insurance.
```

# 6\. Замечания по пункту 5.3.3

Степень уверенности: 95
Условие «ceases doing business» требует географического уточнения. Если не добавить «in CA», фраза может быть истолкована как требование прекратить бизнес вообще, что неверно.
Предложение по устранению:

```markdown
5.3.3) `R` terminates `CA` registration and stops `MT` accrual, provided `C` ceases «doing business» in `CA` under `CA`’s Rev. & Tax. Code § 23101.
```

# 7\. Замечания по пункту 5.3.4

Степень уверенности: 90
Термин «current year» (текущий год) является неточным в налоговом администрировании. Правила VDP оперируют понятием «taxable year» (налоговый год), который у фискальных заявителей может не совпадать с календарным.
Предложение по устранению:

```markdown
5.3.4) `VDP` guarantees a 100% penalty waiver and limits the look-back period to 3 years plus the current taxable year.
```
