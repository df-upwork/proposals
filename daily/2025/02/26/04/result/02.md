After migrating from Desktop to Online, all «jobs» are  converted to «sub-customers».  
Therefore, your problem with the job name on invoices in QuickBooks Online is resolved as follows:  
1) Go to «Sales» → «Customers».  
2) When creating or editing a record, select «Is a sub-customer» and associate it with the desired customer.  
3) Enter the desired name in the «Display name as» field.  
4) Specify the correct settings for «Bill with parent» or «Bill this customer separately».  
When creating a «sub-customer», there is an option «Bill with parent» or «Bill this customer separately».  
If the parent and the «sub-customer» are closely related (for example, a shared account), then select«Bill with parent».  
If the «sub-customer» account is issued separately, then select «Bill this customer separately».  
5) When creating an «Invoice», in the «Customer» field select the required «sub-customer».  
If not selected, only the primary customer's name will appear on the invoice.  
6) Set up the «Invoice» template:  
6.1) Go to «Settings» (gear icon) → «Custom Form Styles».  
6.2) Select the desired «Invoice» template and click «Edit».  
6.3) In the «Design» or «Content» section (depending on the interface version), add or check the field that will display the «sub-customer» name.  
6.4) Save the changes.  
7) If it is necessary to display a name other than «Job Name» or «Sub-customer», then use «Custom fields» and then include this field in the invoice template.
