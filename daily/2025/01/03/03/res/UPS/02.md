На основе требований по отслеживанию:  

1) UPS имеет строгие ограничения на количество запросов к API:  
   - Для производственной среды: 250 запросов в минуту.  
   - Для тестовой среды разработчика: 2 запроса в минуту.  

2) Для вашего объёма в 100,000 запросов:  
   - Вам потребуется обрабатывать примерно 70 запросов в минуту для завершения в течение 24 часов.  
   - Это значительно ниже лимита UPS в 250 запросов в минуту для производственной среды.  

3) Однако есть дополнительные моменты:  
   - Вам нужно явное разрешение от UPS на использование их API для таких больших объёмов.  
   - UPS может потребовать заключить корпоративное соглашение.  
   - Вам следует обратиться в службу поддержки разработчиков UPS, чтобы:  
     - Подтвердить, что ваш сценарий использования разрешён.  
     - Получить необходимые разрешения.  
     - Получить рекомендации по работе с большими объёмами отслеживаний.
