The solution to your task:  
1) В Magento 2 уже существует системный атрибут «Weight».  
Он используется для расчётов параметров доставки: стоимости, сроков и т.п.  
Этот системный атрибут имеет тип «Decimal», а для вашей задачи нужен тип «Dropdown», поэтому системный атрибут «Weight» не получится использовать для вашей задачи.  
2) Поэтому вам нужно создать новый похожий атрибут, с другим именем: например, «Weight Option».  
2.1) Go to the Magento 2 admin panel and navigate to «Stores» → «Attributes» → «Product».  
2.2) Click «Add New Attribute» and create the «Weight Option» attribute.  
2.3) In the «Catalog Input Type for Store Owner» field, select «Dropdown».  
2.4) In the «Advanced Attribute Properties» section, make sure that:  
2.4.1) «Scope» = «Global».  
2.4.2) «Use To Create Configurable Product» = «Yes» (required for using the attribute in configurable products).  
2.4.3) «Visible on Product View Page on Front-end» = «Yes», if you want the attribute to be displayed on the product page.  
2.5) Save the attribute.  
3) Add the necessary values (options) to the attribute:  
3.1) After saving the attribute, open the tab «Manage Options (Values of Your Attribute)».  
3.2) Specify the variants (for example, «500 g», «1 kg», etc.).  
3.3) Save your changes.  
4) Add (or edit) the attribute group in the required Attribute Set.  
4.1) Go to «Stores» → «Attributes» → «Attribute Set».  
4.2) Find the Attribute Set you will use for these products and click on it.  
4.3) Drag the created attribute («Weight Option») from the right column into the desired group on the left so that it is available on the product page.  
4.4) Save your changes.  
5) Configure Simple Products.  
5.1) Make sure that in each Simple Product, the attribute «Weight Option» is filled in with the required value («500 g», «1 kg», etc.).  
5.2) Make sure that all Simple Products belong to the same Attribute Set that contains the attribute «Weight Option».  
6) Create (or edit) a Configurable Product.  
6.1) Go to «Catalog» → «Products» and click «Add Product».  
6.2) In the field «Product Type», select «Configurable Product».  
6.3) Enter the basic product information («Name», «SKU», «Price» — you can set a «starting» one).  
6.4) In the «Configurations» section, click «Create Configurations»:  
6.4.1) Select the attribute «Weight Option» and click «Next».  
6.4.2) Check the required values («500 g», «1 kg», etc.) and click «Next».  
6.4.3) Link the already existing Simple Products (Magento will pick them up automatically if the attributes and SKUs match) or create new ones.  
6.5) If necessary, set up each configuration's images, prices, and quantity.  
6.6) Save the configurable product.  
7) Check the result on the storefront.  
7.1) Go to the page of the created configurable product on the frontend.  
7.2) Make sure that there is now a possibility to select different variants by weight.  
7.3) Make sure that switching between «500 g», «1 kg», etc. works correctly and displays the corresponding prices and stock status.  
8) Make sure that each Simple Product has the correct stock quantity so that customers can only purchase available variants.  
9) If necessary, fill in unique descriptions for each variant (each Simple Product), so that the user sees the necessary information when switching.
