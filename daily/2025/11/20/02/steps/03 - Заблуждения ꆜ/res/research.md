–í —Ö–æ–¥–µ –∞—É–¥–∏—Ç–∞ –≤—ã—è–≤–ª–µ–Ω–æ –Ω–∞–º–µ—Ä–µ–Ω–∏–µ –∫–ª–∏–µ–Ω—Ç–∞ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã–µ —Ç–∏–ø—ã –¥–∞–Ω–Ω—ã—Ö —Å –ø–ª–∞–≤–∞—é—â–µ–π –∑–∞–ø—è—Ç–æ–π (double –∏–ª–∏ float –≤.NET/C#) –¥–ª—è —Ö—Ä–∞–Ω–µ–Ω–∏—è –∏ –æ–±—Ä–∞–±–æ—Ç–∫–∏ —Ñ–∏–Ω–∞–Ω—Å–æ–≤—ã—Ö —Ç—Ä–∞–Ω–∑–∞–∫—Ü–∏–π. –≠—Ç–æ —Ä–µ—à–µ–Ω–∏–µ –∞—Ä–≥—É–º–µ–Ω—Ç–∏—Ä—É–µ—Ç—Å—è "–ø—Ä–æ–∏–∑–≤–æ–¥–∏—Ç–µ–ª—å–Ω–æ—Å—Ç—å—é" –∏ "–ø—Ä–æ—Å—Ç–æ—Ç–æ–π". –î–∞–Ω–Ω—ã–π –ø–æ–¥—Ö–æ–¥ —è–≤–ª—è–µ—Ç—Å—è –∫—Ä–∏—Ç–∏—á–µ—Å–∫–æ–π –∏–Ω–∂–µ–Ω–µ—Ä–Ω–æ–π –æ—à–∏–±–∫–æ–π, –Ω–µ—Å–æ–≤–º–µ—Å—Ç–∏–º–æ–π —Å –ø—Ä–æ—Ñ–µ—Å—Å–∏–æ–Ω–∞–ª—å–Ω–æ–π —Ä–∞–∑—Ä–∞–±–æ—Ç–∫–æ–π —Ñ–∏–Ω–∞–Ω—Å–æ–≤—ã—Ö —Å–∏—Å—Ç–µ–º.

#### **2.1.1. –ü—Ä–∏—Ä–æ–¥–∞ –æ—à–∏–±–æ–∫ –æ–∫—Ä—É–≥–ª–µ–Ω–∏—è –∏ –∏—Ö –Ω–∞–∫–æ–ø–ª–µ–Ω–∏–µ**

–¢–∏–ø—ã —Å –ø–ª–∞–≤–∞—é—â–µ–π –∑–∞–ø—è—Ç–æ–π (float, double) —Ä–µ–∞–ª–∏–∑—É—é—Ç —Å—Ç–∞–Ω–¥–∞—Ä—Ç IEEE 754 –∏ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—Ç —á–∏—Å–ª–∞ –∫–∞–∫ –¥–≤–æ–∏—á–Ω—ã–µ –¥—Ä–æ–±–∏. –§—É–Ω–¥–∞–º–µ–Ω—Ç–∞–ª—å–Ω–∞—è –ø—Ä–æ–±–ª–µ–º–∞ –∑–∞–∫–ª—é—á–∞–µ—Ç—Å—è –≤ —Ç–æ–º, —á—Ç–æ –º–Ω–æ–≥–∏–µ –¥–µ—Å—è—Ç–∏—á–Ω—ã–µ –¥—Ä–æ–±–∏, –∏—Å–ø–æ–ª—å–∑—É–µ–º—ã–µ –≤ —Ñ–∏–Ω–∞–Ω—Å–∞—Ö (–Ω–∞–ø—Ä–∏–º–µ—Ä, 0.1 –∏–ª–∏ 0.01), –Ω–µ –∏–º–µ—é—Ç —Ç–æ—á–Ω–æ–≥–æ –¥–≤–æ–∏—á–Ω–æ–≥–æ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏—è –∏ —Ö—Ä–∞–Ω—è—Ç—Å—è –∫–∞–∫ –±–µ—Å–∫–æ–Ω–µ—á–Ω—ã–µ –ø–µ—Ä–∏–æ–¥–∏—á–µ—Å–∫–∏–µ –¥—Ä–æ–±–∏, —É—Å–µ—á–µ–Ω–Ω—ã–µ –¥–æ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω–æ–π —Ç–æ—á–Ω–æ—Å—Ç–∏.12

–≠—Ç–æ –ø—Ä–∏–≤–æ–¥–∏—Ç –∫ —Å–∏—Å—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∏–º –æ—à–∏–±–∫–∞–º –≤—ã—á–∏—Å–ª–µ–Ω–∏–π. –ü—Ä–æ—Å—Ç–æ–π –ø—Ä–∏–º–µ—Ä: –æ–ø–µ—Ä–∞—Ü–∏—è 100.45 + 25.30 –≤ —Ç–∏–ø–µ double –º–æ–∂–µ—Ç –¥–∞—Ç—å —Ä–µ–∑—É–ª—å—Ç–∞—Ç 125.29999999999998 –≤–º–µ—Å—Ç–æ –æ–∂–∏–¥–∞–µ–º–æ–≥–æ 125.75. –í –º–∞—Å—à—Ç–∞–±–∞—Ö —Ñ–∏–Ω–∞–Ω—Å–æ–≤–æ–≥–æ —Ä–µ–µ—Å—Ç—Ä–∞ (Sales Ledger), –æ–±—Ä–∞–±–∞—Ç—ã–≤–∞—é—â–µ–≥–æ –º–∏–ª–ª–∏–æ–Ω—ã —Ç—Ä–∞–Ω–∑–∞–∫—Ü–∏–π, —ç—Ç–∏ –º–∏–∫—Ä–æ-–æ—à–∏–±–∫–∏ –Ω–∞–∫–∞–ø–ª–∏–≤–∞—é—Ç—Å—è, –ø—Ä–∏–≤–æ–¥—è –∫ —Ä–∞—Å—Ö–æ–∂–¥–µ–Ω–∏—è–º –≤ –±–∞–ª–∞–Ω—Å–∞—Ö –Ω–∞ —Ç—ã—Å—è—á–∏ –¥–æ–ª–ª–∞—Ä–æ–≤. –¢–∞–∫–∏–µ —Ä–∞—Å—Ö–æ–∂–¥–µ–Ω–∏—è –¥–µ–ª–∞—é—Ç —Å–∏—Å—Ç–µ–º—É –Ω–µ–ø—Ä–∏–≥–æ–¥–Ω–æ–π –¥–ª—è –∞—É–¥–∏—Ç–∞ –∏ –Ω–∞–ª–æ–≥–æ–≤–æ–π –æ—Ç—á–µ—Ç–Ω–æ—Å—Ç–∏, –∞ —Ç–∞–∫–∂–µ —É—è–∑–≤–∏–º–æ–π –¥–ª—è —Å–∫—Ä—ã—Ç–æ–≥–æ –º–æ—à–µ–Ω–Ω–∏—á–µ—Å—Ç–≤–∞ ("salami slicing attacks").14

#### **2.1.2. –ò–º–ø–µ—Ä–∞—Ç–∏–≤ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è —Ç–∏–ø–∞ Decimal**

–ï–¥–∏–Ω—Å—Ç–≤–µ–Ω–Ω–æ –≤–µ—Ä–Ω—ã–º —Ä–µ—à–µ–Ω–∏–µ–º –¥–ª—è —Å—Ä–µ–¥—ã.NET —è–≤–ª—è–µ—Ç—Å—è –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ —Ç–∏–ø–∞ System.Decimal (128 –±–∏—Ç). –≠—Ç–æ—Ç —Ç–∏–ø —Å–ø–µ—Ü–∏–∞–ª—å–Ω–æ —Ä–∞–∑—Ä–∞–±–æ—Ç–∞–Ω –¥–ª—è —Ñ–∏–Ω–∞–Ω—Å–æ–≤—ã—Ö –≤—ã—á–∏—Å–ª–µ–Ω–∏–π, —Ç–∞–∫ –∫–∞–∫ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç –¥–µ—Å—è—Ç–∏—á–Ω—É—é –∞—Ä–∏—Ñ–º–µ—Ç–∏–∫—É —Å —Ñ–∏–∫—Å–∏—Ä–æ–≤–∞–Ω–Ω–æ–π –∑–∞–ø—è—Ç–æ–π, –æ–±–µ—Å–ø–µ—á–∏–≤–∞—è —Ç–æ—á–Ω–æ—Å—Ç—å –¥–æ 28-29 –∑–Ω–∞—á–∞—â–∏—Ö —Ü–∏—Ñ—Ä –∏ —Ç–æ—á–Ω–æ–µ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏–µ –¥–µ—Å—è—Ç–∏—á–Ω—ã—Ö –¥—Ä–æ–±–µ–π.15

–û–¥–Ω–∞–∫–æ –¥–∞–∂–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ decimal —Ç—Ä–µ–±—É–µ—Ç —Å—Ç—Ä–æ–≥–æ–π –¥–∏—Å—Ü–∏–ø–ª–∏–Ω—ã. –ü—Ä–∏ —Ä–∞–±–æ—Ç–µ —Å Entity Framework Core (EF Core) –∏ SQL Server –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ —è–≤–Ω–æ –∫–æ–Ω—Ñ–∏–≥—É—Ä–∏—Ä–æ–≤–∞—Ç—å —Ç–æ—á–Ω–æ—Å—Ç—å (Precision) –∏ –º–∞—Å—à—Ç–∞–± (Scale) –¥–ª—è –∫–∞–∂–¥–æ–≥–æ –ø–æ–ª—è —Ç–∏–ø–∞ decimal. –ï—Å–ª–∏ —ç—Ç–æ–≥–æ –Ω–µ —Å–¥–µ–ª–∞—Ç—å, EF Core –º–æ–∂–µ—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –Ω–∞—Å—Ç—Ä–æ–π–∫–∏ –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é, —á—Ç–æ –ø—Ä–∏–≤–µ–¥–µ—Ç –∫ "–º–æ–ª—á–∞–ª–∏–≤–æ–º—É —É—Å–µ—á–µ–Ω–∏—é" (silent truncation) –¥–∞–Ω–Ω—ã—Ö –ø—Ä–∏ —Å–æ—Ö—Ä–∞–Ω–µ–Ω–∏–∏ –≤ –±–∞–∑—É. –ù–∞–ø—Ä–∏–º–µ—Ä, –∑–Ω–∞—á–µ–Ω–∏–µ 100.12345 –º–æ–∂–µ—Ç –±—ã—Ç—å —Å–æ—Ö—Ä–∞–Ω–µ–Ω–æ –∫–∞–∫ 100.12, –µ—Å–ª–∏ —Å—Ç–æ–ª–±–µ—Ü –≤ –ë–î –æ–ø—Ä–µ–¥–µ–ª–µ–Ω –∫–∞–∫ decimal(18, 2), —á—Ç–æ –±–µ–∑–≤–æ–∑–≤—Ä–∞—Ç–Ω–æ —É–Ω–∏—á—Ç–æ–∂–∏—Ç –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—é –æ –¥–æ–ª—è—Ö —Ü–µ–Ω—Ç–∞, –∫—Ä–∏—Ç–∏—á–Ω—É—é –¥–ª—è —Ä–∞—Å—á–µ—Ç–æ–≤ –Ω–∞–ª–æ–≥–æ–≤ –∏–ª–∏ –∫–æ–º–∏—Å—Å–∏–π.16

#### **2.1.3. –ü—Ä–æ–±–ª–µ–º—ã —Å–µ—Ä–∏–∞–ª–∏–∑–∞—Ü–∏–∏ –∏ –ø–µ—Ä–µ–¥–∞—á–∏ –¥–∞–Ω–Ω—ã—Ö**

–ó–∞–±–ª—É–∂–¥–µ–Ω–∏–µ –∫–ª–∏–µ–Ω—Ç–∞ —É—Å—É–≥—É–±–ª—è–µ—Ç—Å—è –Ω–∞ —É—Ä–æ–≤–Ω–µ API. –ü—Ä–∏ –ø–µ—Ä–µ–¥–∞—á–µ –¥–∞–Ω–Ω—ã—Ö —á–µ—Ä–µ–∑ JSON (–Ω–∞–ø—Ä–∏–º–µ—Ä, –≤ —Ñ—Ä–æ–Ω—Ç–µ–Ω–¥ –∏–ª–∏ –≤–Ω–µ—à–Ω–∏–µ —Å–∏—Å—Ç–µ–º—ã) —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã–µ —Å–µ—Ä–∏–∞–ª–∏–∑–∞—Ç–æ—Ä—ã –º–æ–≥—É—Ç –∫–æ–Ω–≤–µ—Ä—Ç–∏—Ä–æ–≤–∞—Ç—å decimal –≤ –æ–±—ã—á–Ω—ã–µ —á–∏—Å–ª–∞ JSON, –∫–æ—Ç–æ—Ä—ã–µ –Ω–∞ —Å—Ç–æ—Ä–æ–Ω–µ –∫–ª–∏–µ–Ω—Ç–∞ (JavaScript) –±—É–¥—É—Ç –∏–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∏—Ä–æ–≤–∞–Ω—ã –∫–∞–∫ double (64-bit float), —Å–Ω–æ–≤–∞ –≤–Ω–æ—Å—è –æ—à–∏–±–∫–∏ –æ–∫—Ä—É–≥–ª–µ–Ω–∏—è.  
–î–ª—è –ø—Ä–µ–¥–æ—Ç–≤—Ä–∞—â–µ–Ω–∏—è —ç—Ç–æ–≥–æ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –ø–µ—Ä–µ–¥–∞–≤–∞—Ç—å –¥–µ–Ω–µ–∂–Ω—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è –ª–∏–±–æ –∫–∞–∫ —Å—Ç—Ä–æ–∫–∏ ("125.75"), –ª–∏–±–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Å–ø–µ—Ü–∏–∞–ª–∏–∑–∏—Ä–æ–≤–∞–Ω–Ω—ã–µ –Ω–∞—Å—Ç—Ä–æ–π–∫–∏ —Å–µ—Ä–∏–∞–ª–∏–∑–∞—Ç–æ—Ä–∞ (–Ω–∞–ø—Ä–∏–º–µ—Ä, FloatParseHandling.Decimal –≤ Json.NET), –≥–∞—Ä–∞–Ω—Ç–∏—Ä—É—é—â–∏–µ —Å–æ—Ö—Ä–∞–Ω–µ–Ω–∏–µ —Ç–æ—á–Ω–æ—Å—Ç–∏.19

### **2.2. –ê—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä–Ω—ã–π –ø–∞—Ç—Ç–µ—Ä–Ω: –°–∞–≥–∞ –ø—Ä–æ—Ç–∏–≤ –î–≤–æ–π–Ω–æ–π –∑–∞–ø–∏—Å–∏**

–ö–ª–∏–µ–Ω—Ç –ø–ª–∞–Ω–∏—Ä—É–µ—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –ø–∞—Ç—Ç–µ—Ä–Ω Saga –¥–ª—è —É–ø—Ä–∞–≤–ª–µ–Ω–∏—è —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã–º–∏ —Ç—Ä–∞–Ω–∑–∞–∫—Ü–∏—è–º–∏ –≤ –º–∏–∫—Ä–æ—Å–µ—Ä–≤–∏—Å–Ω–æ–π –∞—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä–µ P‚Åé, —Å—á–∏—Ç–∞—è –µ–≥–æ —Å–æ–≤—Ä–µ–º–µ–Ω–Ω–æ–π –∑–∞–º–µ–Ω–æ–π –∫–ª–∞—Å—Å–∏—á–µ—Å–∫–∏–º —Ç—Ä–∞–Ω–∑–∞–∫—Ü–∏—è–º. –ü—Ä–∏ —ç—Ç–æ–º –∫–ª–∏–µ–Ω—Ç —É–ø—É—Å–∫–∞–µ—Ç –∏–∑ –≤–∏–¥—É –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ—Å—Ç—å —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏ —Ñ—É–Ω–¥–∞–º–µ–Ω—Ç–∞–ª—å–Ω–æ–≥–æ –ø—Ä–∏–Ω—Ü–∏–ø–∞ –±—É—Ö–≥–∞–ª—Ç–µ—Ä—Å–∫–æ–≥–æ —É—á–µ—Ç–∞ ‚Äî –¥–≤–æ–π–Ω–æ–π –∑–∞–ø–∏—Å–∏ (Double-Entry Bookkeeping).

#### **2.2.1. –ù–µ–æ–±—Ö–æ–¥–∏–º–æ—Å—Ç—å –î–≤–æ–π–Ω–æ–π –∑–∞–ø–∏—Å–∏**

Sales Ledger –Ω–µ –º–æ–∂–µ—Ç –±—ã—Ç—å –ø—Ä–æ—Å—Ç–æ —Ç–∞–±–ª–∏—Ü–µ–π —Ç–µ–∫—É—â–∏—Ö —Å–æ—Å—Ç–æ—è–Ω–∏–π ("–ë–∞–ª–∞–Ω—Å –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—è –ê = $100"). –≠—Ç–æ –¥–µ–ª–∞–µ—Ç –Ω–µ–≤–æ–∑–º–æ–∂–Ω—ã–º –∞—É–¥–∏—Ç –∏–∑–º–µ–Ω–µ–Ω–∏–π. –ü—Ä–æ—Ñ–µ—Å—Å–∏–æ–Ω–∞–ª—å–Ω–∞—è —Å–∏—Å—Ç–µ–º–∞ –¥–æ–ª–∂–Ω–∞ —Ñ–∏–∫—Å–∏—Ä–æ–≤–∞—Ç—å –¥–≤–∏–∂–µ–Ω–∏–µ —Å—Ä–µ–¥—Å—Ç–≤. –ö–∞–∂–¥–∞—è —Ç—Ä–∞–Ω–∑–∞–∫—Ü–∏—è –¥–æ–ª–∂–Ω–∞ –∑–∞—Ç—Ä–∞–≥–∏–≤–∞—Ç—å –º–∏–Ω–∏–º—É–º –¥–≤–∞ —Å—á–µ—Ç–∞ (–î–µ–±–µ—Ç –∏ –ö—Ä–µ–¥–∏—Ç), –∏ —Å—É–º–º–∞ –≤—Å–µ—Ö –¥–µ–±–µ—Ç–æ–≤ –¥–æ–ª–∂–Ω–∞ –≤—Å–µ–≥–¥–∞ —Ä–∞–≤–Ω—è—Ç—å—Å—è —Å—É–º–º–µ –≤—Å–µ—Ö –∫—Ä–µ–¥–∏—Ç–æ–≤.22  
–¢–∞–∫–∞—è —Å—Ç—Ä—É–∫—Ç—É—Ä–∞ –æ–±–µ—Å–ø–µ—á–∏–≤–∞–µ—Ç:

* **–ù–µ–∏–∑–º–µ–Ω–Ω–æ—Å—Ç—å (Immutability):** –û—à–∏–±–∫–∏ –∏—Å–ø—Ä–∞–≤–ª—è—é—Ç—Å—è –Ω–µ –∏–∑–º–µ–Ω–µ–Ω–∏–µ–º –∑–∞–ø–∏—Å–∏, –∞ —Å–æ–∑–¥–∞–Ω–∏–µ–º –∫–æ—Ä—Ä–µ–∫—Ç–∏—Ä—É—é—â–µ–π —Ç—Ä–∞–Ω–∑–∞–∫—Ü–∏–∏.  
* **–ü—Ä–æ—Å–ª–µ–∂–∏–≤–∞–µ–º–æ—Å—Ç—å (Traceability):** –ü–æ–ª–Ω–∞—è –∏—Å—Ç–æ—Ä–∏—è —Ç–æ–≥–æ, –∫–∞–∫ —Å–∏—Å—Ç–µ–º–∞ –ø—Ä–∏—à–ª–∞ –∫ —Ç–µ–∫—É—â–µ–º—É —Å–æ—Å—Ç–æ—è–Ω–∏—é.23  
* **–¶–µ–ª–æ—Å—Ç–Ω–æ—Å—Ç—å:** –í—Å—Ç—Ä–æ–µ–Ω–Ω–∞—è –ø—Ä–æ–≤–µ—Ä–∫–∞ –Ω–∞ —É—Ä–æ–≤–Ω–µ –º–æ–¥–µ–ª–∏ –¥–∞–Ω–Ω—ã—Ö (—Å—É–º–º–∞ —Ä–∞–≤–Ω–∞ –Ω—É–ª—é) –∑–∞—â–∏—â–∞–µ—Ç –æ—Ç –ø—Ä–æ–≥—Ä–∞–º–º–Ω—ã—Ö —Å–±–æ–µ–≤ –∏ –ø–æ—Ç–µ—Ä—å –¥–∞–Ω–Ω—ã—Ö.24

#### **2.2.2. –†–∏—Å–∫–∏ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è Saga –≤ —Ñ–∏–Ω–∞–Ω—Å–∞—Ö**

### **3.2. –¢–µ—Ö–Ω–∏—á–µ—Å–∫–∏–π –ø—Ä–æ–≤–∞–ª –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è Zip-–∫–æ–¥–æ–≤**

#### **3.2.1. –ì–µ–æ–≥—Ä–∞—Ñ–∏—á–µ—Å–∫–æ–µ –Ω–µ—Å–æ–≤–ø–∞–¥–µ–Ω–∏–µ**

#### **3.2.2. –°–ª–æ–∂–Ω–æ—Å—Ç—å Sourcing Rules**

* **Destination-based:** –ë–æ–ª—å—à–∏–Ω—Å—Ç–≤–æ —à—Ç–∞—Ç–æ–≤ —Ç—Ä–µ–±—É—é—Ç –Ω–∞–ª–æ–≥ –ø–æ —Å—Ç–∞–≤–∫–µ –º–µ—Å—Ç–∞ –Ω–∞—Ö–æ–∂–¥–µ–Ω–∏—è –ø–æ–∫—É–ø–∞—Ç–µ–ª—è.
